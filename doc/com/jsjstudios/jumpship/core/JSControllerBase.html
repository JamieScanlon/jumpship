addCommands()method 
protected function addCommands():void

A method that is called to add Commands that this Command can call in the course of its duty's. The format for adding a Command should be as follows:

addCommand ( "getPersonListCommand", GetPersonListCommand );

If "getPersonListCommand" is the same as an event type registered through registerEventDispatcher(), this command will be called automatically as a result of this Controller receiving the event.

public function handleEvent(event:Event):void

The default event handler called by registerEventDispatcher.

This default implementation enables a traditional Command Dispatcher pattern seen in most of the popular frameworks whereby a single event type maps to a single command and parameters are passed through the event itself. By overriding this handler you can define your own implementation. Parameters

registerEventDispatcher()method 
public function registerEventDispatcher(theEventType:String, theEventDispatcher:EventDispatcher):void

Registers an EventDispatcher with this JSControllerBase.

Registers an event dispatcher with this controller. Events registered here will trigger a default event handler that simply calls the command that's associated with theEventType. Registering events in this way enables a traditional Command Dispatcher pattern seen in most of the popular frameworks whereby a single event type maps to a single command and parameters are passed through the event itself. By overriding the default handler, however, you achieve a broader implementation (a many-to-one or one-to-many mapping for instance). Parameters

theEventType:String — A string with the event type to listen for.
 
theEventDispatcher:EventDispatcher — A reference to the EventDispatcher instance.

public final function returnCommandResult(theResultObject:Object):void

Processes the result of a JSCommandBase instance.

This is the default returnFunc given to a new JSCommandBase. This method calls the afterFilter() hook method. This method then calls the original return function supplied in createCommand(). Parameters

theResultObject:Object — An object containing the result information from the command.
