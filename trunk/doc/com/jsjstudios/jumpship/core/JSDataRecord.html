Event object type: com.jsjstudios.jumpship.core.events.DataRecordBindingEvent
DataRecordBindingEvent.type property = com.jsjstudios.jumpship.core.events.DataRecordBindingEvent.UPDATE

Defines the value of the type property of a update event object.

This event has the following properties:

PropertyValue
bubblesfalse
cancelablefalse; there is no default behavior to cancel.
typeIdentifies the type of change that was made.
currentTargetThe object that is actively processing the event object with an event listener.
theRecordIdentifies the record that changed.
theAttributeIdentifies the attribute of the record that changed.
targetThe object that dispatched the event. The target is not always the object listening for the event. Use the currentTarget property to access the object that is listening for the event.
